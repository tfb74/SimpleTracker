# SimpleTracking – App Store Submission Material

Kompletter Stack für die App-Store-Connect-Einreichung. Inhalt dieses Ordners:

- `app-description-de.txt` – deutsche Store-Beschreibung
- `app-description-en.txt` – englische Store-Beschreibung
- `keywords-de.txt` / `keywords-en.txt` – Keywords (max. 100 Zeichen)
- `whats-new-de.txt` / `whats-new-en.txt` – Release Notes für v1.0
- `promotional-text-de.txt` / `promotional-text-en.txt` – kurzer Promo-Text (max. 170 Zeichen)
- `gamecenter-achievements.csv` – die 14 Achievements als Tabelle
- `gamecenter-leaderboards.csv` – die 4 Leaderboards als Tabelle
- `screenshot-specs.md` – welche Screenshots Apple verlangt
- `submission-checklist.md` – komplette Schritt-für-Schritt Liste

## Schnellstart

1. ASC öffnen → App anlegen (Bundle ID `de.baumannheim.SimpleTracking`)
2. `submission-checklist.md` öffnen und durchgehen
3. Copy-Paste der Texte aus den `.txt`-Dateien in die ASC-Felder
4. Screenshots aufnehmen gemäß `screenshot-specs.md`
5. Game Center Setup gemäß `gamecenter-*.csv`
